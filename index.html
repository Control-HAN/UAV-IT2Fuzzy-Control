UAV Control via Interval Type-2 Fuzzy Logic under Cyberattacks.

Abstract

This paper presents an interval type-2 (IT2) fuzzy attitude controller for uncertain unmanned aerial vehicles (UAVs) under hybrid cyberattacks. The UAV dynamics are modeled as IT2 Takagi-Sugeno (T-S) fuzzy systems capturing bounded parametric uncertainties. The control input is stochastically corrupted by switching attacks: (i) sensor-to-controller (S2C) false-data injection distorting state information and (ii) controller-to-actuator (C2A) manipulation causing input saturation. Under the simultaneous effects of these hybrid cyberattacks and physical actuator limits, the control input is subject to nested saturation. To handle the nested saturation under the cyberattacks, a novel generalized set-inclusion (GSI) lemma is introduced. This lemma is utilized in controller design involving general nonlinear functions that satisfy sector nonlinearities. Based on the Lyapunov-Krasovskii functional (LKF) approach, linear matrix inequality (LMI) conditions are derived for controller synthesis. These conditions guarantee closed-loop stability by ensuring that the Lyapunov sublevel set remains within the admissible input bounds. Simulation results demonstrate asymptotic closed-loop stability under hybrid cyberattacks.
